# Dotfiles

## TLDR;

Run this install script to configure the dotfiles.

```sh
git clone https://github.com/wim-de-groot/dotfiles.git
cd ./dotfiles
./install.sh
cd ../
rm -rf dotfiles
```

## Installation

The easiest way to install the dotfiles is to install them via git with a local clone:

```sh
git clone https://github.com/wim-de-groot/dotfiles.git
cd ./dotfiles
./install.sh
cd ../
rm -rf dotfiles
```
